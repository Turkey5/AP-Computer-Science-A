## Commands and Output
This lesson focuses on two commands: System.out.print() and System.out.println().
These are two commands that will both perform the same function, print text to the output terminal.
If we break down these two commands, it is easier to understand how they function.
System is a core class in java, providing many utilities such as the output function we are using.
System.out specifies thag we are using the output field of the System class. 
print() and println() are both methods of System.out, both take a String parameter in the parentheses. That String is what they 
will print to the terminal. 
print() will print the String in the parentheses
println(), also called print line, will print the String in the parentheses, then create a new line (essentially pressing the enter key)
So executing System.out.println() will print whatever string is contained within the parentheses, then create a new line.
